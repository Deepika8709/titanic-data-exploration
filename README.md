# Titanic Data Exploration

This is a beginner Python project that explores the Titanic dataset.

## What this project does
- Loads Titanic data
- Counts survivors
- Finds survival by gender and class
- Calculates average age
- Analyzes children and family size

## Tools used
- Python
- Pandas

## Dataset
Titanic dataset from DataScienceDojo (GitHub)

## How to run
1. Install Python
2. Install pandas:
   pip install pandas
3. Run:
   python titanic.py
